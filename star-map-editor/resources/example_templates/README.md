# Example Templates

This directory contains example template images for testing the Star Map Editor.

You can place your own template images here for easy access when testing.

## Creating Template Images

Template images should be:
- PNG, JPG, or BMP format
- Any size (the editor will handle scaling)
- Ideally showing a galaxy map, star chart, or grid layout

## Example Use Cases

- Hand-drawn galaxy sketches
- Reference maps from other sources
- Grid layouts for alignment
- Background artwork
- Multiple layers of different maps

## Tips

- Use semi-transparent templates (PNG with alpha) for layering
- Start with lower resolution for faster loading during testing
- Use the opacity slider to blend multiple templates
- Lock templates after positioning to prevent accidental changes
