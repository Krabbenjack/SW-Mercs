KEY FEATURES:
─────────────

✓ Map View: Uses full vertical height in both modes
✓ Stats Sidebar: Appears on right when Stats mode is active
✓ Width Control: Sidebar constrained to 250-400px width
✓ Resizable: User can drag splitter handle to adjust widths
✓ Auto-collapse: Sidebar disappears when exiting Stats mode
✓ Stretch Ratio: Map gets 3x more space than sidebar (75%/25%)

IMPLEMENTATION:
───────────────

• Used QSplitter(Qt.Horizontal) instead of vertical layout
• Stats widget moved from above map to beside map
• Splitter automatically handles show/hide behavior
• No changes to Stats widget internal logic
• No changes to other modes or set_mode() function
